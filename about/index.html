Posted on 2019-07-30 Last edited on 2020-07-12

What’s this site?

这里是 Sharzy 的个人博客，用来记录一些个人的发现，以及一些兴致来时写的文章，或者一些我认为比垃圾要略好一些的课程论文。更新随缘。

我是一名 CS 学生，中学时曾有过 MO 背景。虽然宅但也不抗拒运动，对许多 non-trivial 的话题感兴趣，只是常常感慨精力有限，只能浅尝辄止。除了技术和数学以外，对音乐和动画也略知一二，并长期试图涉足各种各样的领域。

在 Github， Bangumi，网易云音乐，豆瓣 可以看到我的一些动态。平时会在我的 telegram channel @sharzy_talk 发送一些吐槽或者一些短篇的文字。

欢迎通过我的邮箱 me@sharzy.in 或者 Telegram 账号 @sharzyl 与我联系。

How is it built?

本站使用使用了 Jekyll 作为网页生成器，使用 markdown 写作，源代码和生成的网页均托管在 GitHub pages 上，在这里可以看到源代码。下面是本站使用的一些第三方库列表：

  • gitalk 提供了评论功能，可以通过 GitHub 账号登陆发表评论。
  • mathjax 提供了数学公式的支持。
  • zooming.js 提供了对于图片放大的支持。

本站支持夜间模式，您可以通过右上角的月亮形图标开启。

如果需要订阅本站的消息，可以通过这个 RSS 链接。

本站的内容，如无特别说明，均使用 CC BY 4.0 协议发布，只要注明来源和修改部分，并且不施加额外的限制，可以将内容分享到任何地方，详情请见 Creative Commons 网站的说明。
